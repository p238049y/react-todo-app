# React Todo App
## 環境構築

```sh
# npm 経由でyarnをインストール
npm install -g yarn
# yarnのバージョンを確認
yarn -v
# yarnの初期設定
yarn init
```

## 初期に追加したもの

```sh
yarn add @tanstack/react-query@4.28.0
yarn add @tanstack/react-query-devtools@4.28.0
yarn add zustand@4.3.6
yarn add @heroicons/react@2.0.16
yarn add react-router-dom@6.10.0 axios@1.3.4
```
